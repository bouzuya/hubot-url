# hubot-url

A Hubot script for fetching the url in user's message.

## Installation

    $ npm install bouzuya/hubot-url

## Usage

    $ # configuration
    $ external-scripts.json
    ["hubot-url"]
    $
    $ # start hubot
    $ hubot
